longshadow.js
=============

Longshadow generator for all divs using box-shadow. Simply plug and play. See demo. 

# Usage

```

longShadow($target, degree, color, length, fade_rate)

```
# Example

```
$(document).ready(function(){
	longShadow($('#demo-div'), 315, '#DDDDDD', 150, 1/150)
})
			
```
